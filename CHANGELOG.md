# CHANGELOG

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project adheres to Semantic Versioning.

## [0.1.1] - 2026-04-18

### Added
- New tree datasets
- Added tree item counts to headings.

### Changed
- Rebuilt and normalized Markdown source dataset
- Re-sorted trees alphabetically within each depth group
- Regenerated all export formats from the corrected source

### Fixed
- Corrected invalid level declarations
- Moved trees to the correct depth groups

## [0.1.0] - 2026-04-17

### Added
- Initial public dataset release
- Markdown source dataset
- Nested Tree JSON export
- Flat Nodes Tree JSON export
- Nested Tree XML export
- Flat Nodes Tree XML export
- Flat Nodes Tree CSV export
- Flat Nodes Tree SQL export
- Basic HTML demo for loading and rendering supported formats
- Trees grouped by depth from 2 Levels up to 8 Levels
